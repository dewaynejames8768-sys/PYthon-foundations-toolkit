## Core Logic & Algorithms
*   **Numerical Comparison Engine:** An utility for identifying minimum values within a dataset, focusing on efficient conditional branching.
*   **Currency/Change Calculator:** Uses modulus (`%`) and floor division (`//`) to decompose total change into precise currency denominations (Dollars, Quarters, Dimes, Nickels, Pennies) with proper grammatical pluralization.

## Data Processing & Transformation
*   **2D Data Parser:** A utility that converts comma-separated string inputs into structured nested lists using list comprehensions. It includes formatting logic for clear, tabular-style data presentation.
*   **Dictionary Mapping Tool:** A file-handling script designed to parse raw lists into structured Key-Value pairs, demonstrating experience in data storage, dictionary manipulation, and file-system interaction.

## Input Validation & User Experience
*   **Seasonal Date Logic:** An robust engine for calendar-based operations. It maps month/day combinations to specific seasons while implementing multi-layered error handling to identify and flag invalid user inputs, ensuring data integrity.
